For live data transfer representation and queue UI please follow the instruction.

1. Go to https://www.rabbitmq.com/install-windows.html and install RabbitMQ
2. Go to http://localhost:15672/ - User name - guest, password - guest.
3. Run MicroServiceRecieve - You can then check in the Queue and the UI managment that a queue was created.
4. Run MicroServiceSend - You can now see the message arriving the queue encrypted.
5. Run MicroServiceRecieve again to see the decryped message.
